# 🧠 OverThinkNoMore

A mental wellness Android application designed to help users manage overthinking, track emotions, and practice healthy coping strategies.

## 📱 Features

### 🤖 AI Chat Assistant

* Chat with an AI-powered assistant powered by Gemini AI.
* Share thoughts and worries in a safe space.
* Receive supportive responses and guidance.

### 😊 Mood Tracker

* Record your daily mood.
* Choose from different mood states:

  * Happy 😊
  * Neutral 😐
  * Sad 😢
  * Angry 😡
  * Tired 😴
* Add personal notes for reflection.

### 🧘 Coping Tools

* Quick activities to reduce stress and anxiety.
* Encourages mindfulness and emotional regulation.

### 📚 Resources

* Access helpful mental health information.
* Learn practical techniques for managing overthinking.

### 📝 Thought Dump

* Freely write down thoughts and worries.
* Organize emotions and clear mental clutter.

---

## 🎯 Purpose

Over Think No More was created to provide users with a simple and accessible tool for:

* Managing overthinking
* Tracking emotional well-being
* Practicing healthy coping strategies
* Encouraging self-reflection
* Supporting mental wellness

## 🚀 Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/over-think-no-more.git
```

2. Open the project in Android Studio.

3. Connect Firebase to the project.

4. Add your `google-services.json` file to the app directory.

5. Sync Gradle and run the application.
